Mission

No single company owns this

OurSQL Foundation exists so the MySQL-compatible ecosystem has a home that no vendor controls.

⚡

Vendor neutral

No single company sets the direction. Decisions happen in the open, with the community at the table.

🔓

Open source first

Permissive licensing matters. We advocate for it and work to keep MySQL-compatible software freely available.

🌐

Ecosystem sustainability

Maintainers burn out. Infrastructure costs money. We fund the work that keeps the ecosystem running.

🤝

Community governance

Membership is open. Decisions are documented. You can see exactly how the foundation operates.

Open Letter · letter.3306-db.org

The MySQL community is asking for change. Add your name.

An open letter to Oracle calling for vendor-neutral governance of the MySQL ecosystem is collecting signatures from developers, DBAs, and organizations. If you run MySQL-compatible databases and want a say in their future, this is a concrete place to start.
